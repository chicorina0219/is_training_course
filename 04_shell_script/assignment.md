# 第4回課題

## 課題3: 空白とワイルドカード

実行結果

1. `$filename`の値は`data 01.txt`である。
  引用符(ダブルクオート)を用いると、空白などを含む文字列をそのまま扱うことができる。
  引用符がないため空白で分割する。
```
$ printf '<%s>\n' $filename
<data>
<01.txt>
```
2. 引用符があるため空白で分割されない。
```
$ printf '<%s>\n' "$filename"
<data 01.txt>
```
3. `$pattern`の値は`*.txt`である。
  `*`: 0文字以上の任意の文字列に一致することを表す。今回の例では、カレントディレクトリにある`*.txt`のパターンに一致するファイル名へ展開する。
  引用符がないためワイルドカードの展開を行う。
```
$ printf '<%s>\n' $pattern
<data 01.txt>
<data 02.txt>
<result.txt>
```
4. 引用符があるためワイルドカードは展開されない。
  `*.txt`が文字列のまま`printf`に渡される。
```
$ printf '<%s>\n' "$pattern"
<*.txt>
```
