# Git commands
## よく使うGitコマンド
- `git status`: 変更されたファイルの確認
- `git add`: ファイルの追加
- `git commit`: コミットする
- `git push`: リモートブランチにプッシュする
- `git pull`: 最新の変更を取得しマージ
- `git branch`: 現在のブランチの確認、作成(brname)、削除(-d brname)
- `git switch brname`: そのブランチへ移動
- `git log`: 履歴の閲覧
- `git diff`: 履歴の比較
